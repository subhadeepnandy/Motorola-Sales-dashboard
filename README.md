📊 Motorola Mobile Sales Performance Dashboard
🧾 Project Overview

This project presents an interactive Motorola Mobile Sales Dashboard designed to analyze sales performance across multiple dimensions such as cities, mobile models, brands, and payment methods.

The dashboard helps stakeholders quickly understand key business metrics, identify trends, and make data-driven decisions.

🎯 Purpose

The goal of this project is to:

Provide a centralized view of sales performance
Track important KPIs like revenue, quantity, and transactions
Analyze customer behavior and payment preferences
Identify top-performing products and regions
🛠️ Tech Stack
Power BI – Data visualization & dashboard creation
Microsoft Excel – Data source & preprocessing
Power Query – Data cleaning & transformation
DAX (Data Analysis Expressions) – Calculated measures and KPIs
📂 Data Source
Data is sourced from Excel files containing:
Sales transactions
Mobile models and brands
Customer ratings
Payment methods
City-level sales data
✨ Features & Highlights
📌 Business Problem

The business lacked a clear and interactive way to:

Track sales performance across regions
Identify best-selling products
Understand customer preferences
Monitor payment trends
🎯 Dashboard Goals
Deliver a single source of truth for sales insights
Enable interactive filtering (Month, Brand, Model, Payment Method)
Improve decision-making with visual insights
